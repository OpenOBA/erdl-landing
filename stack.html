Agent Protocol Stack

Three open, complementary layers for AI agent interoperability

A2A (Google, 2025.04 → Linux Foundation) — HTTP for agents. Handles communication.
ERDL (OpenOBA, MIT) — Traffic rules for agents. Handles behavior.
MCP (Anthropic, 2024.11 → Linux Foundation) — USB for agents. Handles tools.

Complementary, not competing. Complete agent interoperability = MCP + A2A + ERDL.
